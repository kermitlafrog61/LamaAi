### Чтобы запустить проект необходимо:

- Склонировать репозиторий.
```bash
git clone https://github.com/kermitlafrog61/LamaAi.git
```

- Создать .env файл по примеру .env.template указынными в нем данными.

- Запустить docker.
```bash
make run
```

- Или для production.
```bash
make run_dev
```
